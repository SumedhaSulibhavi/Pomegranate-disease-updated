# Pomegranate Disease Detection System

This is a Flask-based web application for detecting pomegranate fruit diseases using a hybrid CNN-ViT model.

## Features
- **Disease Detection**: Upload images to detect diseases like Anthracnose, Bacterial Blight, etc.
- **Chatbot**: AI-powered assistant for agricultural queries.
- **Multi-language Support**: English and Kannada.

## Deployment
This app is configured for deployment on Hugging Face Spaces using Docker.
